# OTUS Wishlist Automation

Автоматизированное тестирование веб-приложения [Wishlist](https://wishlist.otus.kartushin.su)

## Tech Stack

| Категория | Технологии |
|-----------|-----------|
| Язык | Java 11+ |
| Фреймворк | Selenium WebDriver, TestNG |
| Логирование | Log4j2 |
| Управление драйверами | WebDriver Manager |
| Сборка | Maven |
| Паттерны | Page Object, Factory, Singleton |
| Дополнительно | Lombok, Jackson |

## Как запустить тесты

### Все тесты:
```bash
mvn test
